# Data Cleaning & Visualization Project

## Overview
This project demonstrates data cleaning and visualization using the Titanic dataset.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
Titanic Dataset

## Project Workflow
- Loaded the dataset
- Checked dataset information
- Handled missing values
- Removed duplicates
- Removed outliers
- Created visualizations
- Saved the cleaned dataset

## Visualizations
- Age Distribution
- Survival Count
- Survival by Gender
- Correlation Heatmap

## Outcome
This project demonstrates data preprocessing and exploratory data analysis (EDA) using Python.
